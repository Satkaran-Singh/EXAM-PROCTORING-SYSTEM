# 🎓 AI-Based Online Exam Proctoring System

🔗 Repository: https://github.com/Satkaran-Singh/EXAM-PROCTORING-SYSTEM

An AI-powered web application that monitors students during online examinations and detects suspicious activities using computer vision and real-time analysis.

---

## 📌 Overview

With the rapid growth of online education, maintaining exam integrity has become a major challenge. Modern proctoring systems use AI techniques like face detection, gaze tracking, and activity monitoring to ensure fair assessments.

This project provides an automated solution that replaces manual invigilation with intelligent monitoring.

---

## 🚀 Features

* 🎥 Real-time webcam monitoring
* 👤 Face detection & multiple person detection
* 👀 Eye gaze & head movement tracking
* ⚠️ Suspicious activity detection & logging
* 🧪 Secure online exam interface
* ⏱️ Time-based exam system

---

## 🛠️ Tech Stack

* **Backend:** Python (Flask)
* **Frontend:** HTML, CSS, JavaScript
* **AI/ML:** OpenCV, Computer Vision
* **Other Tools:** NumPy, MediaPipe (if used)

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository



